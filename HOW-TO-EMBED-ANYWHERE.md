# How to Embed Anywhere in Your Article

## Two Ways to Add Embeds

### Method 1: Text Editor Button (üìπ) - Embed ANYWHERE
**Use this to insert embeds at any position in your article**

#### Steps:
1. **Click where you want the embed** in your article text
   - Between paragraphs
   - After a sentence
   - At the beginning
   - Anywhere!

2. **Click the Video/Embed button** (üìπ) in the toolbar

3. **Select platform:**
   - Twitter / X
   - Instagram  
   - YouTube
   - Facebook
   - TikTok
   - Spotify
   - Custom

4. **Paste the URL** of the post/video

5. **Click "Insert Embed"**

6. **Embed appears at your cursor position!**

#### Example Workflow:
```
1. Type: "Here's what the president said:"
2. Press Enter
3. Click the Video/Embed button (üìπ)
4. Select "Twitter"
5. Paste tweet URL
6. Click "Insert Embed"
7. Tweet appears right there!
8. Continue typing below the tweet
```

### Method 2: Embed Manager - Embeds at END
**Use this for embeds that should appear at the end of the article**

#### Steps:
1. Scroll to "Social Media Embeds" section (below editor)
2. Click "Add Embed"
3. Select platform and paste URL
4. Add optional caption
5. Save article

#### When to use:
- Related social media posts
- Additional resources
- Supplementary content
- Gallery of embeds

## Comparison

| Feature | Text Editor Button (üìπ) | Embed Manager |
|---------|------------------------|---------------|
| **Position** | Anywhere you click | End of article |
| **Inline** | Yes | No |
| **Between text** | Yes | No |
| **Captions** | No | Yes |
| **Reorder** | Manual | Drag & drop |
| **Best for** | Inline content | End resources |

## Common Questions

### Q: Can I move an embed after inserting it?
**A:** Not easily. Best to:
1. Delete the embed
2. Click new position
3. Re-insert embed

### Q: Can I edit an embed after inserting?
**A:** No, you need to:
1. Delete it
2. Insert new one with correct URL

### Q: Why use Text Editor button vs Embed Manager?
**A:** 
- **Text Editor (üìπ):** For embeds that are part of the story
- **Embed Manager:** For related content at the end

### Q: Can I have both inline and end embeds?
**A:** Yes! Use both methods:
- Text Editor button for inline embeds
- Embed Manager for end embeds

## Best Practices

### 1. Plan Your Embeds
Before writing, decide:
- Which embeds go inline (part of story)
- Which embeds go at end (related content)

### 2. Don't Overuse
- 2-3 inline embeds maximum
- More embeds at end if needed

### 3. Add Context
```
‚úÖ Good:
"The president tweeted about the policy:
[EMBED: Tweet]
This statement caused controversy..."

‚ùå Bad:
[EMBED: Tweet]
[EMBED: Tweet]
[EMBED: Tweet]
```

### 4. Test After Saving
- Save article
- View on frontend
- Check embeds load correctly
- Verify positioning

## Troubleshooting

### Embed appears at wrong position
**Solution:**
1. Delete the embed
2. Click exactly where you want it
3. Make sure cursor is blinking there
4. Then click Video button

### Can't see cursor position
**Solution:**
1. Click in the editor
2. Type a character (like "X")
3. You'll see where cursor is
4. Delete the "X"
5. Now insert embed

### Embed goes to end instead of cursor
**Solution:**
- Make sure you clicked in the editor first
- Editor should have focus (blue border)
- Then click Video button
